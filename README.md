# SriGandha

AI-powered Forest Intelligence Platform for Early Detection and Management of Kole Roga in Sandalwood Trees.

## Tech Stack

- Frontend: React + TypeScript + Tailwind CSS
- Backend: Spring Boot
- AI Service: Python + FastAPI + PyTorch
- Database: PostgreSQL
- Authentication: JWT
- Version Control: Git + GitHub

## Goal

Help farmers and forest officers detect Kole Roga early using AI image analysis, monitor plantations, and provide treatment recommendations.
